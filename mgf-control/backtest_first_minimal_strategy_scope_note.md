# Backtest first minimal strategy scope note

Este run de `Fase 4` queda limitado explícitamente a:
- la `first minimal combined strategy candidate`
- el dataset práctico `MNQ 5m`
- lectura como validación funcional y backtest ligero

No debe interpretarse como:
- validación histórica definitiva;
- promoción del sistema;
- comparación concluyente contra otras variantes;
- evidencia suficiente para robustez.

Razones:
- cobertura corta y reciente del dataset;
- sospecha metodológica de lookahead/sesgo temporal en la integración actual;
- ausencia todavía de una convención de ejecución barra a barra cerrada de forma más estricta.
